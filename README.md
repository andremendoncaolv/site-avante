# Avante — Information Portal

Institutional website for **Avante**, built with HTML, CSS, JavaScript, and PHP. Includes a contact form with email submission.

## Features

- Fully responsive layout
- Informational sections about the organization
- Contact form with server-side email delivery (PHP/`sendEmail.php`)

## Project Structure

```
site-avante/
├── assets/          # Images, fonts, and static resources
├── index.html       # Main entry point
└── sendEmail.php    # Server-side email handler
```

## How to Run Locally

```bash
git clone https://github.com/andremendoncaolv/site-avante.git
cd site-avante
# Open index.html in your browser (PHP features require a local server)
php -S localhost:8000
```
